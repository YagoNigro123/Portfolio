## 💻 Cómo usar este portfolio
Clonar el repositorio:
1.
```bash
git clone git@github.com:YagoNigro123/Portfolio.git
```
2.  
```
cd backend
go mod tidy
npm start
```
3. 
```
cd frontend
npm install
npm start
```
